   	

	what is HTML?

               * HTML stands for Hyper Text Markup Language

	       * HTML is the standard markup language for creating Web pages

	       * HTML describes the structure of a Web page


       what is elements?
 		
	       * HTML element is defined by a start tag, some content, and an end tag.

  			ex:
   			   * <h1>My First Heading</h1>

			   * <p>My first paragraph.</p>
 			   
			   * <a></a>,<ul><ul/>,<ol></ol>

	empty elements?

	       	* HTML elements with no content are called empty elements.
           
			ex:
			   * <br>,<hr>	

	what is Attribute?

	       * HTML attributes provide additional information about HTML elements.

			ex:
 			   class,href,src,width,height,alt

  	Tags:

		* <a>,<b>,<br>,<button>,<i>,<img>,<input>,<label,><span>,<strong>,<sub>,<sup>,<textarea>,
                 <address>,<div>,<footer>,<form>,<h1>-<h6>,<header>,<hr>,<li>,<ol>,<p>,<section>,<table>,<ul>

	Basic structure of html file:

		<html>
			<head>
			  </head>

			<body>
			  </body>
		</html>
